提交次数：1/1

## 题意

将`[1, N]`范围内的数字以某种随机的方式排列并形成一个字符串，从中删除一个字符。给定删除后的字符串和`N`，请输出被删除的字符。（`1 <= N <= 100000`）

## 分析

很简单的一道题。输入`N`之后把`[1, N]`范围内的数字的字符数量都统计一下，然后和字符串里各字符的数量比较一下就可以了。但是，这里的问题是，字符串大概会有多长。因为`N`的最大长度是6，所以字符串长度的上界应该是`600000`。虽然这个界可能比较大，但肯定不应该是`100000`……比赛的时候因此丢掉了若干分。

## 代码

```cpp
#include <iostream>
#include <cstring>
using namespace std;
char str[600005];
int h[10];
int main() {
    ios_base::sync_with_stdio(false); cin.tie(nullptr); cout.tie(nullptr);
    int n;
    cin >> n;
    cin >> str;
    for (int i = 1; i <= n; i++) {
        int x = i;
        while (x > 0) {
            h[x % 10]++;
            x /= 10;
        }
    }
    int m = strlen(str);
    for (int i = 0; i < m; i++)
        h[str[i] - '0']--;
    for (int i = 0; i < 10; i++) {
        if (h[i] > 0) {
            cout << i << endl;
            break;
        }
    }
    return 0;
}
```
